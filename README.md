# 📱 Sheet Sidekick for Foundry VTT

**Sheet Sidekick** is a premium mobile-first utility module for Foundry VTT. It transforms any smartphone or tablet into a dedicated, interactive character sheet—perfect for in-person games where table space is tight, or for keeping your screen clear while playing online.

No more lugging laptops to the physical table. Just pull up your phone, roll your attacks, move your character, and ping the map.

## ✨ Features
* **Native Mobile Feel:** A highly responsive UI tailored for thumbs, complete with safe-area notch padding for modern phones.
* **Battery Saver Mode:** Automatically disables the Foundry canvas (`noCanvas` mode) on mobile devices to save battery life and reduce heat. 
* **Tap-to-Move D-Pad:** A virtual gamepad overlay allows players to move their tokens directly from their phone.
* **Smart Targeting & Pings:** Tap a spell or item, and the module automatically knows if it requires a location. The player's screen will prompt a targeting phase, pushing a ping to the GM's screen (the main map) instantly.
* **Live Target Sync:** View who is currently targeted on the main map directly from the mobile UI.

## 🛠️ Installation
1. Open Foundry VTT and go to the **Add-on Modules** tab.
2. Click **Install Module**.
3. Paste the following link into the "Manifest URL" field:
   `[INSERT YOUR GITHUB RAW MODULE.JSON LINK HERE]`
4. Click Install!

## 💡 How to Use
1. As the GM, open the **Player Access** configuration in the module settings.
2. Check the box next to any player who will be using a mobile device.
3. When that player logs in on their phone, the canvas will be disabled, and they will be presented with the optimized Sheet Sidekick interface!
